## вңЁ Overview

**PlasmaAgent** adalah autonomous AI agent yang berjalan sepenuhnya di atas PostgreSQL sebagai state store. Tanpa Redis, tanpa Celery, tanpa message broker, tanpa local LLM storage вҖ” hanya **Python + PostgreSQL** untuk production-grade autonomous task orchestration.

### рҹҺҜ What Makes PlasmaAgent Different?

| Feature | Traditional Agents | PlasmaAgent |
|---------|-------------------|-------------|
| State Management | Redis/Celery/Files | вң… PostgreSQL only |
| LLM Storage | 10-40GB local models | вң… Zero storage (rule-based + cloud-ready) |
| Task Planning | Manual | вң… Natural language вҶ’ auto-decomposed |
| Error Recovery | Manual intervention | вң… Automatic retry + suggestions |
| Self-Improvement | None | вң… Learns from execution history |
| Concurrency | Message brokers | вң… PostgreSQL `FOR UPDATE SKIP LOCKED` |

## рҹҡҖ Features

### рҹ§  Phase 3: Advanced Intelligence

#### рҹҺҜ Natural Language Understanding
- **рҹ§  Natural Language Task Generation** вҖ” "backup database setiap jam 2 pagi" вҶ’ task otomatis
- **рҹ”Җ Task Decomposition** вҖ” Break complex tasks into sub-tasks dengan dependency graph (DAG)
- **рҹ’ҫ Context Manager** вҖ” Pass context antar sequential tasks (`${prev_task.output}`)
- **рҹӣЎпёҸ Error Recovery** вҖ” Intelligent suggestions saat task gagal dengan 10+ error patterns
- **рҹҺҜ Conditional Execution** вҖ” Execute steps berdasarkan kondisi (`if ${prev.exit_code} == 0`)
- **вҡЎ Parallel Execution** вҖ” Run independent steps concurrently dengan semaphore control
- **рҹ”„ Retry Strategies** вҖ” Exponential backoff, max retries, conditional retry

#### рҹ“Ҡ Self-Improvement Loop
- **рҹ“Ҳ Template Metrics** вҖ” Track success rates, execution times, failure patterns
- **рҹҺЁ Template Evolution** вҖ” Learn dari successful patterns
- **рҹҸ·пёҸ Template Versioning** вҖ” Track template changes over time
- **рҹ”¬ A/B Testing** вҖ” Compare template versions dengan statistical analysis
- **рҹ—‘пёҸ Template Retirement** вҖ” Auto-retire low-performing templates
- **рҹӨ– Auto-Template Generation** вҖ” Create new templates dari user patterns

#### рҹ’Ў Smart Suggestions Engine
- **рҹҺҜ Next Action Recommendations** вҖ” Context-aware suggestions berdasarkan task state
- **рҹ”Қ Similar Task Discovery** вҖ” Find related tasks menggunakan command similarity
- **рҹҡЁ Anomaly Detection** вҖ” Detect suspicious commands (`rm -rf`, `format c:`, SQL injection)
- **вҡЎ Performance Analysis** вҖ” Identify slow commands & optimization opportunities
- **рҹ“Ҡ General Suggestions** вҖ” System-wide recommendations dari task patterns

### вҡЎ Phase 2: Execution Engine

#### рҹҗҡ Shell Executor
- **Async subprocess** dengan real-time output capture
- **Semaphore-based concurrency** control (max 100 parallel)
- **Fail-fast cancellation** dengan proper task cleanup
- **Timeout handling** (max 24 hours per task)
- **Cross-platform** (Windows/Linux/macOS)

#### рҹ“қ Step Management
- Track setiap command dengan status, exit code, duration
- **Execution logs** tersimpan di database (stdout/stderr)
- **Retry mechanism** вҖ” FAILED вҶ’ PENDING, run lagi dari step yang gagal
- **Timeout & Cancellation** вҖ” Configurable per task

### рҹ—„пёҸ Phase 1: Database-Centric Foundation

#### рҹҸӣпёҸ PTSM (PostgreSQL Transactional State Machine)

## рҹ“Ұ Installation

### Prerequisites
- Python 3.13+
- PostgreSQL 18+
- [uv](https://github.com/astral-sh/uv) (recommended Python package manager)

### Quick Install

```bash
# Clone repository
git clone https://github.com/yourusername/plasmaagent.git
cd plasmaagent

# Install dependencies
uv sync

# Setup database
createdb -U postgres plasmaagent

# Run migrations
uv run alembic upgrade head

# Verify installation
uv run plasma doctor
```

## рҹҺ¬ Quick Start

### 1пёҸвғЈ Create a Task

```bash
# Manual task creation
uv run plasma task create \
  --name "Backup Database" \
  --command "pg_dump plasmaagent > backup.sql" \
  --command "echo Backup completed"

# рҹ§  AI-powered task generation (natural language)
uv run plasma task generate --input "backup database postgresql setiap jam 2 pagi" --yes
```

### 2пёҸвғЈ Execute

```bash
# Run task with real-time output
uv run plasma task run --id <task-id>

# Run with streaming output
uv run plasma task run --id <task-id> --stream
```

### 3пёҸвғЈ Monitor & Analyze

```bash
# List all tasks
uv run plasma task list

# Show task details with logs & steps
uv run plasma task show --id <task-id> --logs
uv run plasma task show --id <task-id> --steps

# рҹ“Ҡ View metrics & performance
uv run plasma metrics show
uv run plasma metrics analyze
uv run plasma metrics optimize
```

### 4пёҸвғЈ Smart Suggestions

```bash
# Get AI-powered recommendations
uv run plasma ai suggest --task-id <task-id>

# Find similar tasks
uv run plasma ai similar --task-id <task-id>

# Detect anomalies
uv run plasma ai anomalies --task-id <task-id>
```
